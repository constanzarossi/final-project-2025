Welcome to Cocktail Explorer

This site helps you find cocktails based on an ingredient you have.

How to use

  1. Go to the Search tab.
  2. Type an ingredient (ex: Vodka, Gin, Lime).
  3. If you type part of a word, we’ll show suggestions — click the exact ingredient.
  4. Click View Recipe to see ingredients and instructions.
